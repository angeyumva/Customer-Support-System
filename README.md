# Online Customer Support Ticket System

## Overview
The Online Customer Support Ticket System is a C++ console application developed to help manage customer complaints and service requests for an Internet Service Provider (ISP).

The system uses:
- a **Singly Linked List** to manage open tickets
- a **Stack** to track resolved tickets for audit and review

This project shows how data structures can be applied to solve a real-world problem in customer support management.

## Features
- Add new support tickets
- Resolve existing tickets
- View all open tickets
- View all resolved tickets
- View the most recent resolved ticket
- Record ticket details such as ID, customer name, issue, priority, and timestamps

## Data Structures Used
### Singly Linked List
Used to store and manage open tickets dynamically.

### Stack
Used to keep resolved tickets in **Last In, First Out (LIFO)** order.
